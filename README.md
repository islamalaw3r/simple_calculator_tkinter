# Simple Calculator - Tkinter

آلة حاسبة بسيطة باستخدام Python و Tkinter.

## المهارات المكتسبة
- واجهات رسومية GUI
- استخدام Buttons و Entry
- التعامل مع الأحداث (Hover, Click)
- العمليات الرياضية + الجذر التربيعي وال%
- تحسين التصميم والألوان
